1. Инициализировать GIT:

git init

2. Инициализировать npm пакеты:

npm install

- после выполнения создаётся папка node_modules в корне проекта 

3. Инициализировать webpack :

npx webpack

- после выполнения в корне появляется папка js в корне, а в ней 2 файла  bundle.js  и еще один.



////////////////////////----- Подключение основных файлов ---.///////////////////////////
